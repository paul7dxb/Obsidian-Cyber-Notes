## Commands
| Command | desc |
| --- | --- |
| i | "INSERT" mode |
| esc | command mode |
| h | cursor left |
| l | cursor right |
| k | cursor up |
| j | cursor down |
| w | start of word |
| e | end of word |
| i | insert before cursor |
| I | Insert at beginning of line |
| a | append after cursor |
| A | Append end of line |
| o | make new line under current |
